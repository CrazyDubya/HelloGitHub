内容生成器演示

HelloGitHub 的内容生成器通过以下步骤工作:

模板准备

使用统一的 Markdown 模板，包含变量占位符

{{ "{{ hello_github_content }}" }}
内容注入

将项目内容替换到模板变量中

python make_content.py 112
格式化处理

自动处理链接、图片和格式

统一样式和结构
输出生成

生成最终的 Markdown 文件

ready for publication
脚本使用示例:
# 生成第112期内容
python script/make_content/make_content.py 112

# 生成所有期数内容
python script/make_content/make_content.py all
GitHub 监控机器人

GitHub 机器人自动监控和分析 GitHub 活动，识别热门项目。

监控功能
  • 用户 star 活动跟踪
  • 项目热度评估
  • 实时数据收集
  • API 请求管理
筛选条件
  • 最低 star 数量: 100
  • 活动时间窗口: 1天
  • 排除自己的项目
  • 去重处理
邮件通知格式:
头像 用户名 项目名 时间 Star数
developer awesome-project 2024-01-01 🌟 1,234
注意: 机器人需要配置 GitHub API 密钥和邮件服务器信息才能正常工作。
